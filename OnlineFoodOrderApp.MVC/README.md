# 🍽️ Online Food Ordering Application (In Progress)

A web-based online food ordering application where users can browse restaurants, view menu items, add items to cart, and proceed to checkout.

### 🚀 Tech Stack
- ASP.NET Core MVC
- C# / Entity Framework
- SQL Server
- LINQ
- Bootstrap
- Git / GitHub

### 🔥 Features Implemented
✔ User registration & login  
✔ Restaurant listing  
✔ Add to cart + session based management  
✔ Update & remove cart items  
✔ View cart page with dynamic totals  

### 🛠 Upcoming Features
🔸 Checkout & address input  
🔸 Order placement module  
🔸 Order history page  

### 📁 Project Status: *Work In Progress*
This repository tracks my development progress while practicing ASP.NET Core MVC and database integration.
